--- layout: example.html title: Pre- and post-rendering drawing - Points shortdesc: Add pre- and post-rendering drawing instructions to Point features. docs: >

This example shows how to add canvas instructions before and after OL3 standard drawing phase. Click on plane icons to select them.

Pre- and post-render functions allows to "decorate" an icon with pixel-based elements (for instance, a text 5 pixels below the icon, or smaller icons around the main one). As the feature itself is a parameter of the function, drawings can be based on feature properties.

ol.style.ImageRenderArgs allow easy computation of the extent of the icon.

Beside pre- and post-render functions, there are also foreground-render functions, that are rendered once all other layers have been rendered. Z-order of layers and features are maintained while rendering foreground-render functions.

Note: elements added through pre- and post-render functions are not clickable.
